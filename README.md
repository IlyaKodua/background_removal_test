# background_removal_test

Сервис для удаления фона с изображений на основе модели
[BiRefNet](https://huggingface.co/ZhengPeng7/BiRefNet). Реализован на FastAPI,
упакован в Docker и запускается одной командой через docker-compose.

## Возможности

- `POST /remove-background` — принимает изображение, возвращает PNG с прозрачным фоном

## Запуск
```bash
docker compose up --build
```

## Пример использования
```bash
sh test.sh
```


## Что улучшить для продакшена
Текущий вариант — самодостаточный сервис, удобный для демонстрации и небольших
нагрузок, но нет пакетной обработки. Для реального высоконагруженного прода правильнее вынести инференс на
NVIDIA Triton Inference Server.
